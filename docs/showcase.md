# Showcase

Examples demonstrating rendering and performance characteristics in Kurogane.

* Canvas 2D: CPU-based rendering with batching and cache-friendly data layout
* WebGL2: GPU instanced rendering with large-scale parallelism

Switch between modes in real time to observe differences in throughput and frame stability.

### Controls

* `+` / `-` to increase or decrease instance count
* Switch tabs to toggle between renderers

### Notes

This example focuses on:

* Rendering performance
* Scheduling behavior
* CPU vs GPU scaling characteristics
